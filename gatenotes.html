GATE Exam Notes

1. Engineering Mathematics

Linear Algebra

  • Matrix Rank: 
    rank(A) = Number of non-zero rows in REF
  • Eigen Values: 
    |A - λI| = 0
  • Determinants: 
    Det(A) = Product of Eigenvalues
  • Inverse of Matrix: 
    A⁻¹ = adj(A) / det(A)
  • Systems of Equations: 
    Ax = b has a unique solution if rank(A) = rank(A|b) = n

Calculus

  • Limit Definition: 
    lim x→a f(x) = L
  • Derivative Rule: 
    d/dx [x^n] = n*x^(n-1)
  • Integration Formula: 
    ∫ x^n dx = (x^(n+1))/(n+1) + C
  • Fundamental Theorem: 
    d/dx ∫ f(t) dt = f(x)
  • Taylor Series: 
    f(x) = f(a) + f'(a)(x-a) + f''(a)(x-a)^2/2! + ...

2. Computer Science

Algorithms

  • Time Complexity: 
    Merge Sort: O(n log n)
    BFS/DFS: O(V + E)
  • Recurrence Relations: 
    T(n) = 2T(n/2) + O(n)
  • Sorting Algorithms: 
    QuickSort: O(n^2) worst case, O(n log n) avg case
  • Graph Algorithms: 
    Dijkstra's Algorithm: O((V+E) log V)
  • Dynamic Programming: 
    Fibonacci: O(n) using memoization

GATE Preparation Tips

  • Section Weightage: 
    Technical: 72% | Maths: 15% | Aptitude: 13%
  • Time Management: 
    1-mark Q: 1.5 min | 2-mark Q: 3 min
  • Mock Tests: 
    Attempt at least 10 full-length tests
